==========================
Alyvix Service RESTful API
==========================

This section describes in technical detail the Alyvix Service RESTful API methods.  These
methods are sufficient to allow you to integrate Alyvix Server into any monitoring system.

API calls are based on a root URL of the form ``https://<alyvix_service_ip>:<port>`` where
a GET, PUT, POST or DELETE call such as ``GET /license`` will result in a final URL of the form
``https://<alyvix_service_ip>:<port>/license``.

The included examples are written in Python.

.. warning::

   All API level 0 endpoints (marked "/v0/" through Alyvix Server v1.7.5) are insecure.  They should
   only be used when both caller and server are contained entirely within the same private
   network.
